.Hi! Can you create a program Finding mutual friends between two sets of friends?


# Define two sets of friends
friends_set1 = {"Alice", "Bob", "Charlie", "David"}
friends_set2 = {"Charlie", "Eve", "Frank", "Bob"}

# Find mutual friends
mutual_friends = friends_set1.intersection(friends_set2)

# Print the result
print("Mutual friends:", mutual_friends)

Mutual friends: {'Bob', 'Charlie'}
